# useRandomString

## Introduction

生成随机字符串

## Basic Usage

```ts
import { useRandomString } from '@flypeng/tool/browser';

const randomStr1 = useRandomString();
const randomStr2 = useRandomString(20);
const randomStr3 = useRandomString(20, true);
```
