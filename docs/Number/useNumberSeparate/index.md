# useNumberSeparate

## Introduction

数字根据指定数量进行分割

## Basic Usage

```ts
import { useNumberSeparate } from '@flypeng/tool/browser';
```

## Type Declaration

```ts
/**
 * 数字根据指定数量进行分割
 * @param value
 * @param separateNumber
 * @returns
 */
declare function useNumberSeparate(
  value: number,
  separateNumber?: number = 3,
): string;
```
